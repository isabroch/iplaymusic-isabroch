observe(selectoropt, cbopt, firesOnceopt, rootMarginopt, thresholdopt) → {HTMLCollection}

Source:
  • observe.js, line 24
Use the Intersection Observer (IO) API.
Perform an action on a list of elements that match the selector, when they are within the threshold.
Parameters:
Name Type Attributes Default Description
selector string <optional>
 .lazyLoad The CSS selector for the elements that are targeted, as a string.
cb callback <optional>
 Action to perform on each element targeted by the selector.
firesOnce boolean <optional>
 true If true, will unobserve element after running the callback. False will not unobserve element.
rootMargin string <optional>
 0px 0px 50% 0px Root margin to be used with intersection observer API. Positive values are outset, negative values are inset. Read IO documentation for more information. All values must be px or %, even 0.
threshold number <optional>
 0 How much of the element must be 'observed', from [0 (just a pixel), 1 (the entire element)]. Read IO documentation for more information.
Returns:
List of all the elements being targeted by selector parameter.
